# Political Campaign Site Planner (RU)

## Цель
Помогать быстро спроектировать структуру и контент сайта политической кампании с учётом целей, аудитории и ограничений.

## Когда использовать
- Нужен новый сайт кампании/кандидата/партии.
- Требуется обновить структуру действующего портала.
- Нужно подготовить ТЗ для подрядчика или внутренней команды.

## Шаги работы

1. **Прояснение контекста**
   - Выясни: уровень кампании (федеральный, регион, муниципальный), регион (например, Республика Коми), ключевые темы.
   - Выясни: главная цель сайта (сбор подписей, донаты, информирование, мобилизация волонтёров).

2. **Карта стейкхолдеров и аудиторий**
   - Составь список аудиторий: избиратели, активисты, журналисты, партнёры, наблюдатели.
   - Для каждой: что они хотят сделать на сайте и за сколько кликов.

3. **Карта сайта (sitemap)**
   - Сформируй дерево разделов 2–3 уровней.
   - Отдельно отрази:
     - Страницу кампании / кандидата
     - Новости и повестку
     - Мероприятия
     - Волонтёров и участие
     - Документы и отчётность (по закону)
     - Медиа (фото, видео)
   - Помечай приоритетные разделы (для навигации первого уровня).

4. **Контент-план по разделам**
   - Для каждого раздела: какие типы материалов нужны (новости, лонгриды, FAQ, инфографика).
   - Предложи минимально достаточный набор материалов для запуска (MVP-контент).

5. **Требования к платформе и интеграциям**
   - Если упоминаются Bitrix24, n8n, формы, email-рассылки — явно опиши, какие данные куда летят.
   - Сформулируй базовые требования к аналитике (события, цели, UTM).

6. **Вывод**
   - В конце сформируй:
     - Карта сайта в виде списка.
     - Список обязательных экранов для прототипирования.
     - Список рисков и юридически чувствительных мест, требующих проверки юристом.

## Примеры запросов
- «Спланируй структуру сайта региональной кампании партии, опиши разделы и контент».
- «Сделай карту сайта и контент-план для портала "Зелёная Альтернатива" в Республике Коми».
