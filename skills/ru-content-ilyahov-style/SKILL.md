# RU Content — Ilyahov Style

## Цель
Помогать писать и перерабатывать тексты на русском в информационном стиле: ясно, по делу, с пользой для читателя.

## Когда использовать
- Нужно переписать текст «по-человечески».
- Нужно подготовить текст для сайта, лендинга, инструкции, рассылки.
- Нужно ужать сложную мысль до понятного объяснения.

## Принципы стиля

1. **Польза**
   - Начинай текст с того, что получит читатель.
   - Меньше про автора/организацию, больше про задачу пользователя.

2. **Простота**
   - Короткие предложения.
   - Обычные слова вместо канцелярита.
   - Объясняй термины, если без них нельзя.

3. **Конкретика**
   - Цифры вместо «много/мало».
   - Примеры вместо общих слов.
   - Конкретные действия: что сделать дальше.

4. **Структура**
   - Заголовок → краткий лифлет → основная часть → что делать.
   - Используй списки, чтобы не прятать важное в текст.

## Инструкция для Claude

1. Сначала кратко переформулируй цель текста одним предложением.
2. Потом предложи структуру (заголовки H2/H3).
3. Перепиши или создай текст по этой структуре, соблюдая принципы выше.
4. В конце добавь:
   - 3–5 вариантов заголовков.
   - Короткий вариант текста (анонс/тизер 1–2 предложения).

## Примеры запросов
- «Перепиши этот текст для лендинга в стиле Ильяхова».
- «Сделай информационный текст о запуске нового раздела сайта для избирателей».
